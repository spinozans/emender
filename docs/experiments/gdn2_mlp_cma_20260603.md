## Purpose

Search a local, fair-comparison proxy for NVIDIA GDN-2 at the same 1.3B
local accounting target used by the other CMA runs. The proxy uses the
external GDN-2 mixer plus an official-style SwiGLU MLP (`gdn2-mlp`).

The exact official `gdn2_1.3B` shape has 1,302,638,112 recurrent block
parameters, but lands around 1.418B parameters in this local p50k/tied
embedding harness. This CMA therefore targets 1.3B under the local harness
instead of copying the official shape directly.
